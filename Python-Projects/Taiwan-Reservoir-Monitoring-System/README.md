# 台灣水庫水情監控系統

一個自動化擷取、處理和視覺化台灣水庫即時水情的監控系統。

## 🎯 功能特色
- **自動化資料擷取**：從 21 個主要水庫即時抓取數據
- **資料處理分析**：清洗資料並計算水情統計
- **視覺化圖表**：自動生成水庫蓄水率圖表
- **多格式輸出**：支援 CSV、Excel、PNG 格式
- **錯誤處理機制**：完整的重試機制和日誌記錄

## 📊 擷取的資料項目
- 水庫名稱與位置
- 當前蓄水百分比
- 有效蓄水量（立方公尺）
- 每日水位變化
- 最後更新時間戳記

## 🚀 快速開始

### 環境需求
- Python 3.8+
- Chrome 瀏覽器
- ChromeDriver

### 安裝步驟

```bash
pip install -r requirements.txt
```

### 執行方式

```bash
python main.py
```

## 📁 輸出檔案
- `reservoir_taiwan.csv` - 原始資料（CSV 格式）
- `reservoir_taiwan.xlsx` - 處理後資料（Excel 格式）
- `reservoir_percent_chart.png` - 視覺化圖表

## 🛠️ 使用技術
- **Selenium**：網頁自動化與資料擷取
- **Pandas**：資料處理與分析
- **Matplotlib**：資料視覺化
- **Openpyxl**：Excel 檔案生成

## 📈 系統輸出
系統提供以下統計資訊：
- 所有水庫平均蓄水率
- 最高與最低蓄水率水庫
- 每日水位變化統計
- 根據蓄水率顏色編碼的視覺化圖表

## 🔧 設定配置
無需額外設定 - 使用預設配置即可執行。
