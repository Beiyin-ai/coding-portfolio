# 2235. Add Two Integers

**完成日期：** 2025-12-24  

## 問題描述
給定兩個整數 num1 和 num2，返回它們的和。

## 解法

### 程式碼
```python
class Solution:
    def sum(self, num1: int, num2: int) -> int:
        return num1 + num2
```

### 解題思路
這是最基本的算術運算題目，直接使用加法運算子即可。

### 複雜度分析
- **時間複雜度：O(1)**，只需要一次加法運算
- **空間複雜度：O(1)**，只使用常數空間
