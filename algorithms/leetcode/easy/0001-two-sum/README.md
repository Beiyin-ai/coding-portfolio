# 1. Two Sum

## 解法 1：暴力法（我的第一個解法）

### 程式碼
```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        n = len(nums)
        for i in range(n):
            for j in range(i + 1, n):
                if nums[i] + nums[j] == target:
                    return [i, j]
        return []
```

### 解題思路
- 使用兩層迴圈檢查所有可能的組合
- 第一層迴圈選定第一個數字
- 第二層迴圈從下一個位置開始選第二個數字
- 檢查兩數之和是否等於 target

### 複雜度分析
- **時間複雜度：O(n²)** - 兩層迴圈
- **空間複雜度：O(1)** - 只用了常數空間

雖然這個解法在 n 很大時會很慢，但作為初學者，我認為「先求有，再求好」是很重要的學習策略。
