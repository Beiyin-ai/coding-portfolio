## 📋 專案概述
### 問題背景
台灣中小型工廠普遍面臨「老舊生產設備數據黑箱、能耗不明、難以整合監控」的痛點，
缺乏低成本、易部署的設備監控解決方案。

### 解決方案
開發一個**低成本、非侵入式、即插即用**的智慧化升級套件原型，
透過為傳統設備加裝感測器與邊緣計算單元，實現：
1. **設備狀態監控**（振動預警、溫度監測）
2. **能源消耗可視化**（實時功耗追蹤）
3. **數據集中管理**（統一儀表板顯示）

### 產業價值
1. **預測性維護** - 減少設備意外停機
2. **節能改造依據** - 量化能源消耗，支持ESG環境管理
3. **數位轉型** - 協助傳統工廠邁向工業4.0

## 🏗️ 技術架構

### 1. 感知層 (Edge Sensing)
- **微控制器**：NodeMCU ESP32 (Type-C, CH340, ESP-WROOM-32)
- **感測器模組**：
  - INA219 - 直流電量監測
  - 振動感測器 (SW-420) - 設備健康狀態監測
  - DHT11 - 環境溫濕度監測
  - 人體紅外感測器 - 安全區域監控（可選）
- **受控設備**：
  - 5V直流風扇 - 模擬工業冷卻風扇
  - 微型直流馬達 - 模擬老舊機台主軸

### 2. 通訊與後端層 (Communication & Backend)
- **通訊協定**：Wi-Fi + HTTP POST
- **後端框架**：Python Flask
- **API設計**：RESTful API (`/api/current`)
- **數據格式**：JSON
- **數據處理**：即時接收、暫存、提供歷史數據

### 3. 應用與可視化層 (Application & Visualization)
- **前端技術**：HTML5 + CSS3 + JavaScript
- **圖表庫**：Chart.js - 即時趨勢圖繪製
- **儀表板設計**：工業風格網頁介面
- **顯示內容**：
  - 設備即時功耗
  - 振動狀態監測
  - 環境參數（溫濕度）
  - 歷史數據曲線

### 4. 實體與整合 (Physical Integration)
- **外殼設計**：上掀蓋式透明壓克力展示盒
- **自定義佈局**：內部元件最佳化排列
- **3D列印部件**：使用Tinkercad設計
  - 風扇護網
  - 設備銘牌
  - 感測器支架
- **工業外觀**：專業化視覺呈現
