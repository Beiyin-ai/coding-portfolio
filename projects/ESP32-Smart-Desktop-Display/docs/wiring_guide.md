# ESP32 Type-C 智能桌面顯示器接線指南

## 📋 硬體規格
- **微控制器**: ESP32-WROOM-32 (Type-C 接口版本)
- **顯示器**: SSD1306 OLED 0.96" (128x64, I2C)
- **感測器**: DHT22 溫濕度感測器
- **輸入**: 輕觸按鈕開關
- **通訊**: WiFi 802.11 b/g/n

## 📍 詳細接線表

| ESP32 GPIO | 連接元件 | 腳位說明 | 顏色建議 |
|------------|----------|----------|----------|
| 3.3V | OLED VCC, DHT22 VCC | 電源正極 | 紅色線 |
| GND | OLED GND, DHT22 GND, 按鈕 | 接地 | 黑色線 |
| GPIO21 | OLED SDA | I2C 資料線 | 藍色線 |
| GPIO22 | OLED SCL | I2C 時鐘線 | 黃色線 |
| GPIO4 | DHT22 DATA | 溫濕度資料 | 綠色線 |
| GPIO23 | 按鈕 | 輸入按鈕 | 白色線 |

## 🔧 按鈕接線說明
```
按鈕 ── GPIO23 (上拉輸入)
      │
      └── GND (按下時接地)
```

## ⚠️ 注意事項
1. **DHT22 需要上拉電阻**: 在 DATA 腳和 3.3V 之間接 10kΩ 電阻
2. **電源穩定**: 確保 USB 電源供應充足
3. **短路檢查**: 接線前先用電錶檢查
4. **極性確認**: OLED 和 DHT22 的 VCC/GND 不要接反

## 🎯 接線順序建議
1. 先接所有 GND (黑色線)
2. 再接 3.3V (紅色線)
3. 接 I2C 線 (OLED)
4. 接 DHT22 資料線
5. 最後接按鈕

## 🔍 測試步驟
1. 上傳 MicroPython 韌體
2. 上傳 main.py 和 boot.py
3. 開啟序列埠監控
4. 觀察啟動訊息
5. 測試按鈕功能
