# 常見問題

## DHT22 讀取失敗
- 檢查 DATA 線是否接 GPIO27
- 確認 10K 上拉電阻接在 DATA 和 VCC 之間
- 麵包板中間有沒有通電？

## OLED 不顯示
- 檢查 I2C 位址 (0x3C 或 0x3D)
- 確認 SDA→GPIO21, SCL→GPIO22
- 電源是否穩定？

## INA219 讀取為 0
- 確認 LED 燈條的 5V 電源有接
- 確認電源 GND 和 ESP32 GND 有共地

## 太陽能板讀取為 0
- 檢查分壓電路 (10K+10K) 是否接好
- 確認 GPIO35 有設定為 INPUT

## 功率讀值不穩定
- 確認 LED 固定為同一顏色（粉紅色）
- 確認環境光線穩定（光敏電阻會影響？）
