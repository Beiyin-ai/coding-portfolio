## 🛠️ 工具腳本

所有實用腳本都放在 `scripts/` 資料夾中：
- `check_problems.sh` - 檢查解題統計
- `commit_problem.sh` - 提交特定題目
- `update_leetcode.sh` - 更新整個專案

使用方式：
```bash
cd scripts
./check_problems.sh
```
